# Paint

A BOARD FOR YOU TO DRAW IN PIXEL ART STYLE

-----------------------------------------------------------------------------------------

HOW TO USE:

SPACE BAR - PAINT/ERASE

KEY UP/DOWN/LEFT/RIGHT - FOR MOVEMENT

KEY S - TO SAVE YOUR ART

KEY L - TO LOAD YOUR LAST SAVE

KEY C - TO CLEAR THE GRID
