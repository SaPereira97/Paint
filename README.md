# Paint

A BOARD FOR YOU TO DRAW IN PIXEL ART STYLE

-----------------------------------------------------------------------------------------

HOW TO USE:

KEY UP/DOWN/LEFT/RIGHT - FOR MOVEMENT

SPACE BAR - PAINT/ERASE

1,2,3,4 - CHANGE COLOR

S - TO SAVE YOUR ART

L - TO LOAD YOUR LAST SAVE

C - TO CLEAR THE GRID

ESC - TO CLOSE THE APP
